# Number 11 Berlin Website

This is the official website for **Number 11 Berlin**, an immersive sports arena and premium hospitality venue at Sony Center, Berlin.

## Tech Stack

*   **Framework:** Next.js 15 (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **Icons:** Lucide React (optional/ready for integration)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

*   `app/`: Contains all the routes and page logic.
    *   `page.tsx`: Home page.
    *   `experience/`: The Experience / Venue page.
    *   `suites/`: Private Suites page.
    *   `food-market/`: Food Market page.
    *   `sponsors/`: Sponsors & Partnerships page.
    *   `investors/`: Investors Overview page.
*   `components/`: Reusable components like `Navbar` and `Footer`.

## Design Notes

*   **Theme:** Dark, premium, immersive (Black, Dark Grays, Yellow/Gold Accents).
*   **Fonts:** Geist Sans / Geist Mono (Next.js default, can be swapped for custom fonts).
*   **Images:** Placeholders are currently used. Replace them in the code where indicated with `[Placeholder: ...]`.
